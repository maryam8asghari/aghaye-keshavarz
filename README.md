# آقای کشاورز 🌾

پلتفرم مدیریت پرونده و مشاوره کشاورزی هوشمند

## معرفی پروژه

**آقای کشاورز** یک پلتفرم وب موبایل‌محور (PWA) برای مدیریت پرونده‌های کشاورزی و ارائه مشاوره تخصصی به کشاورزان ایرانی است. این سیستم با هدف اتصال کشاورزان به راهبران محلی و متخصصان کشاورزی طراحی شده است.

## قابلیت‌های اصلی

### ۱. نقشه مبتنی بر نقش
هر نقش کاربری فقط موجودیت‌های مرتبط خود را در نقشه مشاهده می‌کند:

| نقش | موجودیت‌های قابل مشاهده |
|---|---|
| **کشاورز** | راهبران محلی |
| **راهبر محلی** | متخصصان |
| **متخصص** | آزمایشگاه‌ها، نهالستان‌ها، فروشگاه‌ها |

### ۲. سیستم پرونده
- ثبت پرونده جدید با عکس و توضیحات
- تحلیل هوش مصنوعی (AI Triage)
- تخصیص خودکار راهبر محلی
- بازدید میدانی و گزارش‌نویسی
- ارجاع به متخصص در صورت نیاز
- ارائه توصیه تخصصی
- پایش نتیجه و بازخورد

### ۳. آرشیو پرونده‌ها
- مشاهده تمام پرونده‌های ثبت شده
- جستجو بر اساس نوع محصول
- فیلتر بر اساس وضعیت پرونده
- پیگیری روند پیشرفت پرونده

## نقش‌های کاربری

### کشاورز 👨‍🌾
- ثبت پرونده جدید
- مشاهده راهبران محلی در نقشه
- دسترسی به آرشیو پرونده‌های قبلی
- پیگیری وضعیت پرونده‌ها

### راهبر محلی 👨‍💼
- مشاهده متخصصان در نقشه
- مدیریت پرونده‌های تخصیص داده شده
- انجام بازدید میدانی
- ارائه گزارش بازدید
- ارجاع به متخصص

### متخصص 👨‍🔬
- مشاهده آزمایشگاه‌ها و نهالستان‌ها در نقشه
- بررسی پرونده‌های ارجاعی
- ارائه توصیه تخصصی
- مشاهده نتیجه نهایی

## تکنولوژی‌های استفاده شده

| لایه | تکنولوژی |
|---|---|
| فریمورک | Next.js 16 (App Router) |
| زبان | TypeScript |
| رابط کاربری | React 19 |
| استایل | Tailwind CSS v4 |
| مدیریت state | Zustand |
| نقشه | Leaflet + React-Leaflet |
| انیمیشن | Framer Motion |
| آیکون‌ها | Lucide React |
| PWA | Serwist |
| تقویم | date-fns-jalali |

## ساختار پروژه

`
src/
├── app/                    # صفحات Next.js
│   ├── page.tsx           # صفحه خانه و انتخاب نقش
│   ├── map/               # نقشه
│   ├── case/              # پرونده‌ها
│   │   ├── new/           # پرونده جدید
│   │   ├── [id]/          # جزئیات پرونده
│   │   └── archive/       # آرشیو پرونده‌ها
│   ├── dashboard/         # داشبورد
│   └── advisor/           # پروفایل مشاور
├── components/            # کامپوننت‌ها
│   ├── MapView.tsx        # نقشه با فیلتر نقش
│   ├── Navbar.tsx         # نوار پایین
│   ├── AdvisorCard.tsx    # کارت مشاور
│   └── ...
├── data/                  # داده‌های Mock
│   ├── cases.ts           # پرونده‌ها
│   ├── advisors.ts        # مشاوران
│   ├── sponsors.ts        # حامیان
│   └── specialist-locations.ts  # مکان‌های متخصص
└── store/                 # مدیریت state
    └── useAppStore.ts     # Zustand store
`

## راهنمای نصب و اجرا

### پیش‌نیازها
- Node.js 18 یا بالاتر
- npm یا yarn

### نصب وابستگی‌ها
`ash
npm install
`

### اجرای پروژه
`ash
npm run dev
`

پروژه در آدرس http://localhost:3000 اجرا می‌شود.

### بیلد برای تولید
`ash
npm run build
npm run start
`
